# Patient Portal Prototype

This repository contains only the mobile-first patient portal web prototype.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

If port `3000` is already in use, Next.js will print a different local URL such as `http://localhost:3001` or `http://localhost:3002`.

## Main route

- `/` redirects to `/mobile`
- `/mobile` is the patient portal prototype

## Verification

```bash
npm run lint
npm run build
```
